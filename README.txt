# 🖥️ Linux System Monitor (Python)

A simple Linux-based system monitoring tool built using Python.  
This project helps users view system processes, memory usage, CPU usage, and manage processes directly from the terminal.

---

## Features

-  Show running processes
-  Kill a process using PID
-  Display memory usage
-  Display CPU usage
-  Auto-refresh mode (updates every few seconds)
-  Input validation and error handling

---

##  Technologies Used

- Python
- Linux Commands (`ps`, `free`, `top`, `kill`)
- subprocess module
- os module
- time module

---

## ▶️ How to Run

1. Open terminal in project folder  
2. Run the following command:

```bash
python3 monitor.py
